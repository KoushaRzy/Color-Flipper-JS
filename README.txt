Random Color Generator

A simple web application that allows users to change the background color of the page to a random color or a custom color entered into an input field.
Features

    Generate a random HEX color for the page background.
    Apply the color you write into the input field to change the background.
    The input field is cleared on page reload.

How to Use

    Click "Change To Color that I wrote" to set the background color to the custom color you enter.
    Refresh the page to clear the input field.

Installation

    Clone the repository.
    Open index.html in your browser.

Technologies Used

    HTML
    CSS
    JavaScript
